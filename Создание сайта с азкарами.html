javascript
const azkar = {
  morning: [
    {
      arabic: "اللّهـمَّ أَنْتَ رَبِّـي لا إلهَ إلاّ أَنْتَ...",
      translit: "Аллахумма анта Рабби, ля иляха илля Ант...",
      translate: "О Аллах, Ты мой Господь, нет божества, кроме Тебя...",
      audio: "https://www.islamcan.com/audio/dua/dua1.mp3"
    }
  ],
  evening: [
    {
      arabic: "أَمْسَيْنا وَأَمْسَى المُلْكُ للهِ...",
      translit: "Амсайна ва амса аль-мульку лиллях...",
      translate: "Мы вступили в вечер, и власть принадлежит Аллаху...",
      audio: "https://www.islamcan.com/audio/dua/dua2.mp3"
    }
  ]
};

function showAzkar(time) {
  const container = document.getElementById("azkarContainer");
  container.innerHTML = "";

  azkar[time].forEach(zikr => {
    const card = document.createElement("div");
    card.className = "azkar-card";

    card.innerHTML = `
      <p class="arabic">${zikr.arabic}</p>
      <p class="translit">${zikr.translit}</p>
      <p class="translation">${zikr.translate}</p>
      <audio controls src="${zikr.audio}"></audio>
    `;

    container.appendChild(card);
  });
}

// Темная тема
document.getElementById("themeToggle").onclick = () => {
  document.body.classList.toggle("dark");
};

🛠️ Шаг 1: Регистрация на GitHub

  1. Перейдите на https://github.com.

  2. Нажмите Sign up (Зарегистрироваться).

  3. Введите:

    • Имя пользователя (например, azkar-brother)

    • Адрес электронной почты

    • Пароль

  4. Следуйте инструкциям для завершения регистрации.

📁 Шаг 2: Создание репозитория

  1. После входа в аккаунт нажмите на значок "+" в правом верхнем углу и выберите "New repository".

  2. Введите:

    • Repository name: azkar-site

    • Description: Сайт с утренними и вечерними азкарами

  3. Выберите Public и установите галочку на "Add a README file".

  4. Нажмите Create repository.

📂 Шаг 3: Загрузка файлов сайта

  1. В открывшемся репозитории нажмите "Add file" > "Upload files".

  2. Перетащите или выберите файлы вашего сайта:

    • index.html

    • style.css

    • script.js

  3. После загрузки нажмите "Commit changes".

🌐 Шаг 4: Включение GitHub Pages

  1. Перейдите во вкладку "Settings" вашего репозитория.

  2. В меню слева выберите "Pages".

  3. В разделе "Source" выберите ветку main и папку /root.

  4. Нажмите "Save".

  5. После сохранения появится ссылка на ваш сайт, например: https://ваше_имя.github.io/azkar-site/.
